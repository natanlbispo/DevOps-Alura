## Vagrant 

Através de arquivo ficar as configuraçõeas do ambiente com virtualização 

--- 
Instalar 
- VitrualBox
- Instalar Vangrant


--------------------------------

### Comandos:

Cria ambiente
```  vagrant init hashcorp/precise64 ``` 

Inicialização
``` vagrant up  ```

Status das máquinas
``` vagrant status ```

Subir a máquina 
``` vagrant up ```

Pare a máquina virtual
``` vagrant halt```

--
Conhexão 
``` vagrant ssh```

Sair da conexão 
``` exit ```
